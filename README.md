# AI Fundamentals Journey

A structured, project-driven learning log documenting the process of building 
foundational knowledge in machine learning, AI systems, and automation.

## What's in here

**Daily Learning Notes:** Each day includes specific learning goals, resources used, 
and key concepts absorbed. Not a diary — focused technical notes on what was learned 
and why it matters.

**Weekly Projects:** Four capstone projects demonstrating hands-on understanding:
- Week 1: Python automation scripts solving real problems
- Week 2: Machine learning fundamentals (linear algebra, probability, gradient descent)
- Week 3: Building and training ML models + RAG chatbot architecture
- Week 4: Full AI system integration — personal assistant pipeline

## Why this exists

Three years in design taught me systems thinking. Now building the technical depth 
to understand how intelligent systems actually work — not just use them.

This repo is the learning process made visible.
